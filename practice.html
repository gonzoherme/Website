Python

If you'd like to get started with Python, here are some tutorials that we reccomend:

  • Python Crash Course by Eric Matthes : we definitely recommend getting started with this book. It walks you through Python basics, starting from scratch (text editors, how to set up Python on your operating system, variables, loops...). What makes the book even better is the fact that after learning the basics of the language, it helps you put them to the test with fun and practical projects, such as building your own game, generating data with Python, downloading data, using django... Suprisingly, that is not all. Despite all of what has been mentioned previously, this book has still got more to offer. It teaches you how to handle Git, Stack Overflow and other tools to facilitate your work as a programmer. (link to the book)

  • YouTube : YouTube is still a fairly reliable source of information when it comes to tutorials. Here are some links to tutorials that you might consider useful: (Tutorial by freeCodeCamp.org) (Another tutorial, by "Programming with Mosh")

    • Octave / Machine Learning

    Since we thought many of you would be interested in looking into machine learning, we have listed below what we think is one of the best machine learning courses out there. It is Andrew Ng's Stanford Machine Learning Course, offered for free in Coursera. We have actually enrolled and completed the 11 weeks of this course. So it is out of prior experience, that we definitely reccomend taking this course.

    However, if you don't know anything at all about machine learning, or you haven't seen much about it, we do think it would be a better strategy to look into the basics of this topic before jumping into the course. Although it is accessible for beginners, having a bit of background on the subject will surely help you get through the course smoothly.
